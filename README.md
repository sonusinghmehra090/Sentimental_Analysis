# Sentimental_Analysis
### A robust, deep learning-based sentiment analysis model trained on the classic IMDB Movie Reviews Dataset. Featuring over 4 million trainable parameters, this model is fine-tuned to classify movie reviews as either highly positive or negative with strong generalization capabilities.

# 🚀 Project Overview
#### Sentiment analysis is a fundamental Natural Language Processing (NLP) task. This project leverages a custom deep neural network architectures (LSTM) to understand the semantic nuances of user-generated movie reviews.

##### Language: Python

##### Deep Learning Framework: PyTorch

##### Data Manipulation: NumPy, Pandas
